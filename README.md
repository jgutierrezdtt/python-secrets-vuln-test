# python-secrets-vuln-test

Repositorio de ejemplo con archivos Python que contienen secretos incrustados en el código (para pruebas).

Instrucciones:

1. Ejecuta `python3 generate_vulns.py` para generar los archivos vulnerables y `vulnerabilities.csv`.
2. Revisa los archivos en `vuln_code/` y `tests/`.
3. Usa el CSV con el script de creación de issues para crear issues en GitHub.
